# Catalogo de Documentacion

Este directorio concentra la documentacion funcional del dashboard operativo.

## Contenido

- `especificacion_tecnica_tablero_oc_abiertas.md`: especificacion objetivo del tablero, con arquitectura, ETL, frontend, NL2SQL y QA.
- `dashboard_operativo.md`: explica la estructura de la pantalla y que muestra cada bloque.
- `glosario_columnas.md`: define las columnas fuente mas importantes del modelo.
- `lineage_metricas.md`: mapea cada indicador a sus columnas, reglas y funciones de codigo.

## Documento tecnico principal

La documentacion detallada de formulas y KPIs se encuentra en:

- `../metricas_kpi.md`

## Uso recomendado

1. Leer `especificacion_tecnica_tablero_oc_abiertas.md` como documento rector de implementacion.
2. Leer `dashboard_operativo.md` para entender la pantalla.
3. Leer `metricas_kpi.md` para entender las formulas.
4. Usar `glosario_columnas.md` cuando haya dudas sobre columnas del Excel.
5. Usar `lineage_metricas.md` para trazabilidad tecnica.
